# Wisdom High School - Rwimi Website

This is the website for Wisdom High School - Rwimi, a licensed private secondary school in Bunyangabu District, Uganda. Created using HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Hero Slider**: Auto-rotating image slider with manual controls
- **Navigation**: Sticky header with mobile-friendly hamburger menu
- **Quick Links Section**: Easy access to key university information
- **About Section**: Information about the university
- **Video Tour Section**: Prominent video call-to-action
- **Programs Section**: Showcase of academic programs
- **Events Section**: Upcoming events listing
- **Newsletter Subscription**: Email subscription form
- **Course Search**: Search functionality for courses
- **Footer**: Comprehensive footer with links and contact information

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript (ES6+)**: Interactive features and dynamic content
- **Font Awesome**: Icons
- **Google Fonts**: Poppins, Playfair Display, and Roboto fonts

## Key Features Implemented

1. **Hero Slider**: Auto-rotating slideshow with indicator controls
2. **Mobile Menu**: Hamburger menu for mobile devices
3. **Smooth Scrolling**: Smooth scroll behavior for anchor links
4. **Scroll Animations**: Fade-in animations on scroll
5. **Form Handling**: Newsletter and course search form interactions
6. **Scroll to Top**: Button appears when scrolling down
